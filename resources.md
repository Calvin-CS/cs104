# Resources

## Development Environment

### Coder
The course can use a cloud-based coding environment through **Coder**, so you do not need to install everything locally to get started.

- Log in at [Calvin's Coder instance](coder.cs.calvin.edu) and open a workspace
- Your workspace includes Python and the tools needed for class work
- [What is Coder?](https://coder.com/docs/about) — remote development environments in a browser
- **Tip:** your files persist within your workspace — save often and keep your work organized

## Version Control

### GitHub
Lab assignments are distributed and submitted through **GitHub Classroom**.

- [Sign up for GitHub](https://github.com/signup) — use your `.edu` email to get the [Student Developer Pack](https://education.github.com/pack) for free
- Each lab will give you a GitHub Classroom link that creates a personal repo for your work
- Push your completed notebook to your repo to submit

**Getting started with Git & GitHub:**
- [GitHub's own quickstart guide](https://docs.github.com/en/get-started/quickstart)
- [Git cheat sheet (PDF)](https://education.github.com/git-cheat-sheet-education.pdf)
- Key commands you'll use:

```bash
git add .
git commit -m "completed lab"
git push
```
