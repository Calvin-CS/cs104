# Modules, Dependencies, and Virtual Environments

### CS-104 · Week 7

---

## Modules

- A module is a Python file you can import.
- Modules help organize code across files.
- Imports let you reuse code cleanly.

---

## Dependencies

- Some projects need external packages.
- External packages must be installed before import.
- Good projects document what they need.

---

## Virtual Environments

- A virtual environment isolates project packages.
- It helps avoid version conflicts.
- `venv` is a standard Python tool for this.

---

## End-of-Course Goal

- Read simple Python code confidently
- Write small multi-file programs
- Debug common errors
- Keep your programming environment organized
