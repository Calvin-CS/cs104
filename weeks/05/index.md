# Week 5: Loops

## Learning Objectives

By the end of this week, students will be able to:

- Use `for` loops to repeat a task over a known collection or range of values
- Use `while` loops when repetition depends on a changing condition
- Update variables correctly inside a loop
- Recognize and avoid common loop errors such as off-by-one mistakes and infinite loops
