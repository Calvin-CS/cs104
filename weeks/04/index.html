Week 4: Conditionals

Learning Objectives

By the end of this week, students will be able to:

  • Write if, elif, and else statements to control program flow
  • Use boolean expressions to decide which branch of code runs
  • Trace a program with multiple conditional paths
  • Test edge cases that exercise each branch of a program
