# Data Types

### CS-104 · Week 3

---

## Core Python Types

- `int`
- `float`
- `str`
- `bool`

---

## Why Types Matter

- Types affect what operations are valid.
- The same input can behave differently depending on type.
- Good programmers check assumptions early.

---

## Type Conversion

- `int("5")` converts text to a number.
- `str(42)` converts a number to text.
- Not every conversion works automatically.

---

## Strings

- Strings store text.
- Indexing and slicing let us inspect pieces.
- Concatenation combines strings.
